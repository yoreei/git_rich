# Git Rich

![git_rich](https://github.com/yoreei/git_rich/assets/5225386/abc9bb2f-0e4d-4d49-b62a-6188fd12bf67)

#Controls:

- You can skip dialog by clicking on it.
- In the  top left corner you have HELP buttons.
- In the bottom left corner you have a health bar.
- Don't forget we have special button for rage quit!
- Or just use Alt + F4 for a regular exit!
- This game was made for the Hardcor Game Jam, which is a part of the Global Game Jam. Made within a 36h time period.

# How to setup the project

* Download Git from [here](https://git-scm.com/)   
* Open your command-line interpreter and type:    
```
git clone git@github.com:yoreei/git_rich.git
```
* Mouse right click on TopDownARPG.uproject -> "Generate Visual Studio Project Files"
